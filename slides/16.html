  • Testing the console command is a straightforward approach.
  • Since all the console commands are registered as a service we need to fetch the service in pass in command tester class
  • So this is not a complex thing to test the commands
  • For additional testing in tables, Console has a test class (Symfony\Component\Console\Tests\Helper) https://github.com/symfony/console/blob/7.1/Tests/Helper/TableTest.php where we can refer additional tests.
