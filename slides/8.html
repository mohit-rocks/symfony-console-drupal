symfony/console

  • Component of the Symfony framework
  • Easy input output handling
  • Rich terminal features like colors, progress bars, etc.
  • Symfony/Console is a component of the Symfony framework that provides tools for building command-line applications in PHP
  • It provides easy input and output handling with various command building blocks.
  • We can also use a console to build.
    • Single command applications.
    • It has various command output building blocks that makes results easier to manage and display
  • We can use additional features like various formatting options like table helper, questions helper, etc for various formatting options.
    • Apart from that, it has multiple components to format output like
    • Title
    • Section
    • Listing
    • Comment
    • Table etc
  • Since commands are services you can run service from regular code like call the service directly etc.
    • Allows to run certain commands from UI
    • Example of running command from the UI
